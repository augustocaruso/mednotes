# MedNotes Study

Use this skill for public educational review of medical-study material.

Keep the output focused on:

- key concepts
- reasoning steps
- common pitfalls
- questions the student should be able to answer

Do not process patient-identifying information. If the input appears to contain
private patient data, stop and ask for a de-identified version.
